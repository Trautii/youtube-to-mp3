# youtube-to-mp3
Will download youtube video(s) and convert them to MP3s. You can choose between single video/song, playlists, etc.

Depends on:
yt-dlp

Recommended additional install:
nodejs, ffmpeg



Usage:
youtube-to-mp3.sh [--mode single|playlist|auto] [--sanitize auto|on|off] [--out DIR] <URL|FILE>



Example:
Download a single mp3 from a URL:
./youtube-to-mp3.sh https://www.youtube.com/watch?v=-YVlgXN7spw&list=RD-YVlgXN7spw&start_radio=1
